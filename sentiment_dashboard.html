Sentiment Dashboard - Template

About This Template

This is a template for a personal sentiment dashboard that visualizes conversation history from Claude and ChatGPT exports.

The full dashboard includes:

  • Sentiment and wellbeing tracking over time
  • Word cloud analysis per month
  • People mention tracking
  • Key events timeline
  • Drama triangle analysis
  • LLM-generated insights

Setup Instructions

1. Export your data from Claude.ai and ChatGPT

2. Place exports in ./llm data/

3. Run the data processing script:

node process_data.js

4. (Optional) Run LLM analysis for deeper insights:

node llm_analysis.js

5. Start a local server and open the dashboard:

npx serve .
# Open http://localhost:3000/sentiment_dashboard.html

Data Privacy

This dashboard processes your personal conversation data locally. No data is sent to external servers (except for optional LLM analysis which uses your own API key).

Keep your private_data/ folder secure and never commit it to public repositories.
